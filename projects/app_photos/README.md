# Readme

The default picture path is: `/maixapp/share/picture`, you can also use `app::get_picture_path()` to get the picture path.

Below is a reference for the format of saving images:

```shell
/maixapp/share/picture
    /2024-01-02
        0.jpg
        1.jpg
        2.jpg
    /2024-02-03
        0.jpg
        1.jpg
        2.jpg
```
